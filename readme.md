### Twiter clone Backend Service

## Install Dependency

```bash
$ npm install
```
## Run app

#### Developmenet environment

install nodemon to use this command ` npm i -g nodemon`

```bash
$ npm run dev
```

#### Complile Bable & build

```bash
$ npm run build
```

#### Production environment

```bash
$ npm run start
```
